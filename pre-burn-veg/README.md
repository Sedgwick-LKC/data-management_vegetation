# Pre-Burn Vegetation Code

Surveys of the vegetation communities before a burn is an important part of establishing baseline conditions.

## Script Explanations

- `pre-burn_ground-cover-01_wrangle.r` -- ingest 'raw' ground cover survey data from WildNote app and get it into preferred format
- `pre-burn_oak-survey-01_wrangle.r` -- same process for oak survey data
- `pre-burn_understory-survey-01_wrangle.r` -- same process for understory data
